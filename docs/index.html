GroupBulletinBoard

LFG/LFM Request Aggregator for World of Warcraft 3.3.5a

⬇ Download v1.6.0 📂 View Source

🔍 Smart Detection

Automatically detects LFG/LFM requests from chat channels and organizes them by dungeon.

🎮 Ascension Support

Dedicated filter panel for custom Ascension dungeons: Vault of the Inquisition and Road to De' Other Side.

🌍 Multi-Language

Supports English, German, Russian, French, and Chinese dungeon detection.

🔔 Notifications

Get sound or chat alerts when groups are looking for your dungeons.

⚡ Performance

Optimized with UI throttling and caching for smooth gameplay even with heavy chat spam.

🎨 Customizable

Filter by dungeon, level range, heroic/normal mode. Fold/unfold dungeons to focus on what matters.

⌨️ Slash Commands

/gbb - Open main window
/gbb config - Open settings
/gbb reset - Reset position
/gbb notify chat on - Chat alerts
/gbb notify sound on - Sound alerts
/gbb help - Show all commands
